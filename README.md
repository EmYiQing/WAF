# WEB应用防火墙（WAF）

- Java编写，基于SpringBoot框架和拦截器

- 暂时只有SQL注入模块，后续会增加XSS，文件上传等

- 分为两种安全级别，HIGH模式会防止一切SQL注入的可能性；LOW模式尽量地考虑不影响到业务
